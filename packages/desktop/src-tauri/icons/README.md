# Icons placeholder

Drop these before running `tauri build`:

- 32x32.png
- 128x128.png
- 128x128@2x.png
- icon.icns (macOS)
- icon.ico (Windows)

Design: red folder glyph with link/chain overlay, brand red `#D8393D` on dark card. Export from the same source as SyncPad's icons for visual family.

Quick bootstrap:

```
tauri icon path/to/logo-1024.png
```
